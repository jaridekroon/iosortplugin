Sort the external memory

The goal of this exercise is to sort the elements located in external memory. You can do the following actions:

  • Move a block from external memory to an empty internal memory block.
  • Move a block from internal memory to an empty external memory block.
  • Sort the elements in internal memory.

You can reset the memory to the initial configuration at any point. Once you have sorted all elements and put them back in external memory, press the "Submit" button.
